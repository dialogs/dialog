Интеграционные возможности

Доступность для сторонних сервисов
Чтение/запись данных для сторонних сервисов в Kafka. Внутренние сервисы доступны по API (gRPC/GraphQL).
Внешняя/отчуждаемая аутентификация
Для аутентификации пользователей в Dialog вы можете подключить LDAP, использовать внешнего SSO-провайдера, RBAC систему или свой коннектор.
SSO-провайдер
Dialog может выступать как SSO провайдер для внешних сервисов, ботов и мини-приложений.
